# 🌦️ Weather App

A simple Weather App built using **HTML, CSS, and JavaScript**. It fetches real-time weather information using the **WeatherAPI** and displays it in a clean user interface.

## 🚀 Features
- 🔍 Search weather by city name
- 🌡️ Current temperature
- ☁️ Weather condition
- 💧 Humidity
- 🌬️ Wind speed
- 🤒 Feels like temperature
- ⏳ Loading state
- ❌ Error handling for invalid city names
- ✅ Input validation (minimum 3 characters)

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Fetch API)
- WeatherAPI

## 📁 Project Structure

```
weather-app/
│── index.html
│── style.css
│── script.js
│── README.md
```

## ▶️ How to Run
1. Clone this repository.
2. Open the project folder.
3. Add your WeatherAPI API key in `script.js`.
4. Open `index.html` in your browser.
